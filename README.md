# JS6 kisha

## <a name="Installation"></a>Installation ##

First install  [node.js](https://nodejs.org). Then install dependencies:
* `npm install`

To start raining:
* `npm start`

Then open [http://localhost:8080/](http://localhost:8080/). The browser will automatically update when you save.
